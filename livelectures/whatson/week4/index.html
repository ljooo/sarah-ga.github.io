bad logos

final branding + logo design tips

  • a website is always marketing something: product, person, concept
  • every decision has a purpose
  • there are no rules, only conventions

typefaces vs fonts

  • Helvetica vs Helvetica, 12px, bold, black

warning! use display typefaces properly

  • used to highlight or bring attention to an element
  • scripts, handwritten, “wacky” typefaces
  • loads of personality (in a bad way)

how to choose typefaces + style fonts

  • limit number of typefaces per page to 3
  • let the main font shine; simplify the rest
  • avoid similar fonts
  • mix complementary fonts
  • introduce scale to create hierarchy, contrast + balance
  • remember white space

final font tips

  • experiment! trial and error, see what works best
  • how does it feel?
