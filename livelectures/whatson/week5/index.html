The universal selector

  • *{ } is the universal selector
  • applies style properties to all individual elements
  • replaces inherited style properties and default 'initial values'
  • blocks inheritance

Baby's First Reset

  • *{
    margin:0;
    padding:0;
    box-sizing:border-box;
    }
  • margin:0; overrides intial auto values
  • padding:0; overrides intial auto values
  • box-sizing:border-box; controls total width of HTML elements
