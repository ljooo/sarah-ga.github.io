fixed (or static) layout

  • size of objects unaffected by browser size or device

fluid (or liquid) layout

  • uses % to allow the objects to resize fluidly
  • sizing based on parent element

adaptive layout

  • uses fixed widths + media queries
  • resizes based on device

responsive layout

  • uses fluid widths + media queries
  • resizes based on device

mobile up vs desktop down

  • mobile up = main CSS controls mobile design
  • desktop down = main CSS controls desktop design

how to begin

  • make sure you're using semantic elements
  • check your nests
  • validate your code with inspect element or the validator
  • add fluidity to parents first, then children
  • make note of breaking points with device mode
  • finish with media queries
